Legendary Motosport

Este projeto foi meu primeiro projeto de site, criado inteiramente para testar minhas habilidades com HTML e CSS, inspirado pela loja dentro do jogo GTA 5, tentei recriar o visual unico da loja do jogo, enquanto aprendia sobre o CSS.

Esse projeto teve inicio em 2023, sendo feito apenas para a versão desktop, devido a minha falta de conhecimento na época não cheguei a fazer nenhum script em JavaScript, nem a fazer responsividade no site para versão mobile. Mas ainda pretendo voltar a esse site e finalizar

Após a "finalização" desse projeto, principalmente na época e com os conhecimentos que eu tinha, me senti bem satisfeito pelo progresso, porém não pelo resultado final, como dito anteriormente ainda preciso realmente finalizar esse site e ajustar algumas coisas, mas com certeza esse primeiro projeto foi o projeto mais divertido que ja fiz até agora
